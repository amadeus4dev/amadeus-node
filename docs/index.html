Amadeus

The Amadeus client library for accessing the travel APIs

new Amadeus(options: Object)
Parameters
options (Object = {}) a list of options
Name Description
options.clientId string the client_id used to authenticate the API
options.clientSecret string the client_secret used to authenticate the API
options.logger Object (default console) an optional console compatible logger that accepts log , error and debug calls. Defaults to a plain console.
options.hostname string (default 'test') the name of the host ( test or production ) to make an API call against.
options.customAppId string (default null) a custom App ID to be passed in the User Agent to the server.
options.customAppVersion string (default null) a custom App version to be passed in the User Agent to the server.
options.http Object (default https) an optional Node HTTPS compatible client. Defaults to the standard Node HTTPS client.
Properties
client (Client) : A wrapper client that handles all the API calls
version (number) : The version of this API client
